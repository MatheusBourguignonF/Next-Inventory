# 🛒 Next-Inventory

O **Next Inventory** é uma aplicação web desenvolvida com foco em prática de desenvolvimento front-end moderno, simulando uma plataforma de e-commerce no estilo Mercado Livre e AliExpress, onde produtos de diferentes empresas são exibidos em uma interface moderna e interativa.

O projeto foi construído com atenção a boas práticas de componentização, tipagem forte e construção de interfaces escaláveis.

---

# 🚀 Tecnologias utilizadas

Este projeto foi desenvolvido utilizando:

- Next.js
- React
- TypeScript
- Tailwind CSS
- Shadcn/ui

---

# 🔐 Funcionalidades

A aplicação conta com:

Página inicial com listagem de produtos
Exibição de produtos em cards com informações resumidas
Modal com detalhes completos do produto
Acesso a informações da empresa responsável pelo produto dentro do modal
Modal com detalhes da empresa selecionada
Interface moderna, responsiva e componentizada com shadcn/ui
